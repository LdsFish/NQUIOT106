# NQUIOT106
物聯網通識專題-Smart Home Bot
106學年第一學期第九組

Bot運作方式  
![image](NQUIOT106/image/botcallled.jpg)
```
開燈>>>亮blue LED、導通繼電器並回應燈泡已經開啟
關燈>>>亮red LED、關閉繼電器並回應已經關閉
閃爍>>>LED交替閃爍、並回應一閃一閃亮晶晶
全關>>>關閉所有LED燈、並回應close it all
```
